
# C Basic Programming Tasks

This repository contains 8 beginner-level C programs demonstrating basic control structures, arithmetic operations, and character handling.

## List of Programs

| File | Description |
|------|--------------|
| Task1_EvenOdd.c | Checks whether a number is even or odd |
| Task2_AbsoluteValue.c | Calculates absolute value of an integer |
| Task3_SubtractNumbers.c | Subtracts smaller number from larger |
| Task4_PositiveNegative.c | Checks whether a number is positive or negative |
| Task5_EqualityCheck.c | Checks if two integers are equal or not |
| Task6_SimpleCalculator.c | Simulates a simple calculator |
| Task7_CharacterType.c | Determines if a character is uppercase, lowercase, or symbol |
| Task8_GradeSwitch.c | Displays grade based on marks using switch statement |
